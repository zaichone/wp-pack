== Description ==

Automatically converts all Gravity Form labels into HTML5 placeholders. Includes Modernizr to add placeholder support to Internet Explorer.

== Installation ==

1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is this for? =

By default, Gravity Forms does not include support to use placeholders. A placeholder is the equivalent of a label, but is found inside the field (input or textarea) instead of beside it. This plugin takes all Gravity Form labels and automatically converts them to placeholders.
